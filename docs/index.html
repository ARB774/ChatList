Возможности

Что умеет ChatList

Сравнение моделей

Отправляй один промт сразу в несколько нейросетей и сравнивай ответы в едином окне.

Сохранение результатов

Храни полезные ответы и промты в локальной SQLite-базе с быстрым доступом из интерфейса.

Работа с релизами

Скачивай установщик и следи за версиями прямо со страницы проекта на GitHub Pages.

Установка

Как установить и запустить

  1. Скачай `ChatListApp_Setup_1.0.0.exe` или открой страницу релиза.
  2. Установи приложение через мастер установки.
  3. Создай файл `.env.local` рядом с `ChatListApp.exe` или в `C:\Users\%USERNAME%\AppData\Local\ChatList\`.
  4. Добавь нужные ключи API и запусти `ChatList`.
OPENROUTER_API_KEY=ваш_ключ
HUGGINGFACE_API_TOKEN=ваш_ключ
